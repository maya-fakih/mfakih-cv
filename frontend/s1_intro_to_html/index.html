HTML Basics:


Main Tags

  • <html> tag

    The root element that wraps all HTML content.
    It tells the browser where the document starts and ends.

  • <head> tag

    Contains metadata and document information.
    This part isn't visible to users but is crucial for the browser.

    • <h1> tag

      Largest heading, most important for the page title.

    • <h2> tag

      Secondary heading, used for major sub-sections.

  • <title> tag

    Sets the browser tab title.
    Important for user navigation and bookmarks.

  • <body> tag

    Contains all the visible page content like text, images, and buttons.

  • <center> tag

    Centers content on the page.
    Note: We usually use CSS for this nowadays!

  • <u> tag

    Underlines the enclosed text for visual emphasis.

  • <ul> tag

    Creates an Unordered List (bullet points).

  • <li> tag

    Defines a List Item inside a list.

  • <p> tag

    Defines a Paragraph. Browsers add a bit of space around it automatically.

  • <strong> tag

    Indicates that the text has strong importance or urgency.
    It renders as bold by default.
