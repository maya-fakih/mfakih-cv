HTML basics:

Main Tags:

  • <html> </html>
    The <html> tag tells the browser that this is an HTML document. The </html> tag tells the browser that the HTML document has ended.

  • <head> </head>
    The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

  • <title>
    The <title> element defines the title of the document, which is shown in a browser's title bar or on the page's tab.

  • <body>
    The <body> element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

  • <div>
    The <div> tag defines a division or a section in an HTML document.

  • <span>
    The <span> tag is used to group inline elements or text in a document.

  • heading tags:
    HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading.
    • <h1>
    • <h2>
    • <h3>
    • <h4>
    • <h5>
    • <h6>

  • <ul>
    The <ul> tag defines an unordered (bulleted) list.

  • <li>
    The <li> tag defines a list item.

  • <center>
    The <center> tag is used to center-align text.

  • <br>
    The <br> tag inserts a single line break (new line or enter).

  • <p>
    The <p> tag defines a paragraph...

  • <img>
    The <img> tag is used to embed an image in an HTML page. It is a self-closing tag and contains attributes such as 'src' (source) and 'alt' (alternative text).

  • <table>
    The <table> tag defines a table.
